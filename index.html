How AI Helped Me Build This Site

Intro AI Tools Pros & Cons Conclusion

Introduction

This project explores how AI tools can support beginners in programming. I used tools like ChatGPT to help write and understand code while building this very site.

Top AI Tools for Programming

  • ChatGPT: Explains code and answers questions.
  • GitHub Copilot: Auto-suggests code inside editors like VS Code.
  • Codeium: Free alternative for code completions.

Pros and Cons of AI in Programming

Pros

  • Fast help with bugs and errors
  • Good for learning new concepts
  • Can write sample code quickly

Cons

  • Sometimes gives incorrect code
  • Not good for full projects alone
  • Doesn't always explain well

Conclusion

AI helped me not just by writing code, but by teaching me how to understand it. I now feel more confident editing and improving simple websites.
